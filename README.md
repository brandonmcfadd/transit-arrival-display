A React Node.js app that makes calls to the CTA's API and displays the results via a display page. 

Display Page: localhost:3000/

Required Inputs:
• stpid or mapid - Must be in comma separated list 

Optional Inputs: 
• walkTime - Place an integer in this input argument to hide results less than this value 

Example:
• localhost:3000/?walkTime=0&mapid=40330,40380
